# Front end boilerplate

### Installation
You need Gulp installed globally:
<pre>
$ npm install --global gulp-cli
</pre>

#### - Main process
<pre>
$ git clone https://github.com/gilren/front-end-starter.git project name
$ cd project name
$ yarn
$ gulp
</pre>

##### - Available tasks
<pre>
$ gulp
$ gulp prod
$ gulp clean
$ gulp copy
</pre>
